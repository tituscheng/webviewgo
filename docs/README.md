# webviewgo Documentation

This directory contains project documentation, roadmaps, and gap trackers for the `webviewgo` library.

## Files

| File | Purpose |
|------|---------|
| [`ROADMAP.md`](ROADMAP.md) | Phased development plan with milestones and estimated effort |
| [`GAPS.md`](GAPS.md) | Detailed inventory of all known gaps, stubs, and TODOs |
| [`PLATFORM_MATRIX.md`](PLATFORM_MATRIX.md) | Feature-by-platform support matrix |

## Updating These Docs

When you fix a gap or implement a feature:

1. Update the status in `GAPS.md`
2. Move the item to the "Completed" section
3. Update `PLATFORM_MATRIX.md` to reflect the new capability
4. Update `ROADMAP.md` phase progress

> **Note:** These docs are living documents. Keep them current as work progresses.
